# CrowdLens

A web app to share photos from any event.

## Project Structure

This is a monorepo containing:

- **apps/frontend** - React + Next.js frontend application
- **apps/backend** - Cloudflare Workers backend with Wrangler

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Development

Run the frontend:

```bash
npm run dev:frontend
```

Run the backend:

```bash
npm run dev:backend
```

### Build

Build all applications:

```bash
npm run build
```

### Linting

```bash
npm run lint
```
